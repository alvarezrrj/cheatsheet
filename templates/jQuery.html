The symbol $ is an alias for jQuery which in turn is a property of the window object and contains all jQuery methods and properties.

To ensure all javaScript code is run only when the DOM is loaded on to the browser, use the jQuery .ready() method. Put all your code inside it:

$(document).ready( () => {
  // jQuery code

To check whether a selection contains any elements, don't do if ( $("div.foo)) {}. this will always evaluate to true because the selection always returns an object and empty objects are truthy. Instead check for the length of the returned object being zero or positive (if ( $("div.foo").length){}).

The eq(int) method reuturns the element in the given index as a jQuery object. This means it has methods like .html() and .after(). Create new elements

Use $("<tagName>inner html goes here</tagName>, {attribute1: value1, attribute2: value2, ... , attributeN: valueN}) to create new elements and then use .insertAfter() or appendTo() to chuck it into the DOM. Take a look at the docs to find out more about creating, moving and deleting elements.

CSS

It is best practise to follow the convention of separation of oncerns. This refers to keeping files to do one specific thing, in this case, it refers to leaving the css in a css file and using javaScript to add and remove classes to the elements instead of modifying css properties in a javaScript file.

To achieve the above, use the methods .addClass('className'), removeClass('className') or .toggleClass('className) which adds a class to the element if it hasn't got it and removes it if it has it.

Access and modify CSS properties with $("div.foo").css("propertyName", "value"). Give the method an object of property:value pairs to set multiple properties at once or pass it only a property name to use it as a getter. Rememeber to camelCase property names that include a hyphen.

The .animate() method takes in two arguments, an object of css properties and values (end values) and the time in miliseconds you want the transition of those porperties to take.

Events

Use the .on("eventNAme", callback(event)) method to add event listeners. Certain events have a shorthand notation: .click(), .focus(), .blur() and .change(). Check out the docs for a full list of properties and methods that the event object contains. In adition to the event object, the event handling function has access to the DOM element on which the event was triggered via the keyword this.
