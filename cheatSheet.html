HTML

Some important definitions

HTTP
HyperText Trasfer Protocol. It's the language the two computers trying to send stuff to each other use to comunicate.
TCP
Transmission Control Protocol. Manages the channels between computers that are opened every time a computer makes a request from a server.
URL
Uniform Resource Locator. It's the adress that contains the files.
IP
Internet Protocol
Category Tag Element Attributes Values Notes
Head <meta> Metadata 1)charset,
2)name,
3)content		 1)UTF-8,
2)description, keywords, viewport, author		 Use the meta tag inside the head tag to include all this things. See the reference here.
Visual formatting <i> Italics Makes the text appear in italics.
<b> Bold Makes the text appear in bold.
<details> Details widget Creates a widget that needs to be toggled open in order to see the information contained in it. See how to use it here.
<figure> Figure Used for sticking together a piece of media and its caption through the <figcaption> tag. When moving the figure with CSS, its caption will go with it.
<hr> Horizontal Rule Draws a line across the screen.
Semantics <em> Emphasis Emphasises text and makes it appear in italics.
<strong> Strong Emphasises text and makes it appear in bold. For very important text.
<q> Quotation Encloses text in quotes.
<header> Page header Encloses the page's header containing the company logo, name and navaigation manu.
<nav> Navigation menu Used for enclosing the navigation menu. Very important for SEO and page readers.
<main> Main content Encloses the page's main content (sandwiched between header and footer).
<section> Section Used for splitting the main content of te page into sections.
<article> Article Contains independent, self-contained content from a forum post, blog post or news story. It can be nested inside a Section and vice versa.
<aside> Aside Used for information that is related to its surrounding content. Usually placed as a sidebar.
<footer> Page footer Encloses the page's footer containing contact info, site map, links to privacy policy, etc.
<thead> Table header Envelops and defines the table's header.
<tbody> Table body Envelops and defines the table's main content.
<tfoot> Table footer Envelops and defines the table's footer (conclusions/totals/etc).
<time> Time Use this tag when indicating dates and times for SEO. See how here.
<blockquote> Block Quotation cite Set the cite attribute's value to the url of the quote Nest a p element for the quoted text and a <cite> element for the quote. Reference.
Lists <ol> Ordered List type
  • a: lowercase letters
  • A: uppercase letters
  • i: lowercase roman
  • I: uppercase roman
 Creates a list of <li>s with ordered numbers.
<ul> Unordered List Creates a list of <li>s with bullet points.
<li> List item value An integer indicating the current value of the list item. Following lis will number themselves from here.
<dl> Definition List Creates a list of titles (<dt>) and definitions (<dd>).
<menu> Menu Creates a list of <li>s.
Table <table> Table Creates a table.
<tr> Table row Creates a table row.
<td> Table definition rowspan |
colspan		 Takes a numeric value for the cell to ocupy that number of cells. Gives content to a cell.
<caption> Table caption style caption-side: bottom Is a child of the table element and defines its caption (default position: top).
<colgroup> Column group Defines a group of columns by taking the element <col> as children. More info.
Media <audio> Audio controls |
autoplay		 It takes a <source> tag as a child which takes the 'src' attribute for linking the content
<embed> Embed src It is a self closing tag and is used for embeding any sort of media. It can also be used to embed html into html.
<video> Video width | height | controls | autoplay Use the 'controls' attribute for the browser to add basic play controls. Use source tags as children to provide different sources in case of incompatibility. Write the alternative text (to display if video is not available) between the opening and closing tags. reference.
<img> Image src | alt | title | height | width The value of the 'title' attribute diplays on mousover. the 'alt' attribute allows you to select the text to display when the image is not available. Determine the size in pixels.
<picture> Picture Allows you to set different sources for an image depending on view or device. Use <source> tags as children with media and srcset attributes for specifying screen conditions and file source respectively. Use an <img> tag as the last child for browsers that do not support the picture element or screens that don't match the given conditions. Reference.
<svg> Scalabe Vector Graphics Defines vector-based graphics in XML format. W3Schools tutorial.
Links <a> anchor element target "_blank" to open in new tab.
Forms <input> Form input pattern Use the pattern attribute to force the input field to match a regular expression.
<label> label: a caption for an item in a UI for set for = to the id of the element it is labeling Labels make items readable by screen readers. Set the innerHTML to the description of the item.
<fieldset> fieldset Wrap your form items in a pair of fieldset tags to draw a box around them. Find out more.
